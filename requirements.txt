# REQUIRED EXTERNAL SERVICES:
#
# 1. RabbitMQ Server
#    Purpose: Message broker for inter-process communication
#    Installation:
#      Windows:
#        - Install Erlang first: https://www.erlang.org/downloads
#        - Install RabbitMQ: https://www.rabbitmq.com/download.html
#        - Or use Chocolatey: choco install rabbitmq
#      Docker (any OS):
#        - docker run -d --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3-management
#      Linux:
#        - sudo apt install rabbitmq-server (Ubuntu/Debian)
#        - sudo systemctl enable rabbitmq-server
#        - sudo systemctl start rabbitmq-server
#    Verification: 
#      - Service should be accessible on localhost:5672
#      - Management UI (if enabled): http://localhost:15672
#
# 2. SSH Server (on remote HPC system)
#    Purpose: Remote command execution and file transfers
#    Requirements:
#      - SSH daemon running on remote server
#      - SSH private key file for authentication
#      - Network connectivity to remote server
#      - User account with appropriate permissions
